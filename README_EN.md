# Misuzu Music

[简体中文](README.md) | English

A local music player.

## Download

- Windows / Linux / macOS: download installers from [GitHub Release](https://github.com/MCDFsteve/MisuzuMusic/releases/).
- iOS: get Misuzu Music on the [App Store](https://apps.apple.com/us/app/misuzu-music/id6755328428?l=zh-Hans-CN).

## Features

- Import local music; create playlists and a play queue
- Song details support Markdown and display furigana for Japanese kanji
- Load local LRC and embedded lyrics; automatically fetch lyrics and cover art online
- Lyrics experience: tap a line to seek, glow effect, hide-lyrics toggle, refined translation display, tighter interaction layout
- Desktop lyrics and Apple Music–inspired design; iPad layout matches the desktop UI
- Cross-platform and lightweight; supports mounting WebDAV / iCloud

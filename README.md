# Misuzu Music

一个本地音乐播放器。

## 功能：

- 导入本地音乐
- 允许创建歌单
- 允许加载本地歌词lrc文件
- 支持显示日语汉字的假名注音
- 支持对没有歌词和封面海报的音乐自动联网获取歌词和封面海报
- 轻量
- 跨平台
- 仿Apple Music设计风格
